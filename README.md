Добавил поддержку wysihtml5 редактора в модуль Wysiwyg для Drupal 7

Версия Jquery должна быть не ниже 1.7.1 (https://drupal.org/project/jquery_update) 

По умолчанию отключена поддержка изображений и все кнопки редактора на Русском языке!

Проект экспериментальный и не гарантирует стабильную работу. Используйте на свой страх и риск!
